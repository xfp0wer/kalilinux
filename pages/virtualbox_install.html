VirtualBox Install

    What is VirtualBox

    VirtualBox is a application that allows you to run a different type of computer (Windows, MacOS, Kali Linux) or the same type of computer using you computer

    In this image you can see that my main operating system (OS) is Windows 11, but using the app VirtualBox so I can also run kali Linux at the same time

    How to install VirtualBox

    To install VirtualBox you have to go to this link https://www.virtualbox.org/wiki/Downloads then click on your OS. For me its windows so i click Windows Host. Once its done open the file and folow the instuctions

    How to use Kali Linux on VirtualBox

    Once installed open VirtualBox. Once opened you will see a screen that looks something like this.

    Now your screen won't show Kali Linux, because you have to install it to the VirtualBox app. Inorder to do that go to this link https://www.kali.org/get-kali/#kali-installer-images Now you have to find out what bit you computer uses, unless you have apple, but after that click the download icon for the image that says INSTALLER.

    Once the Kali Linux file is installed DO NOT OPEN IT! Go back to VirtualBox and click new. Enter a name and click next.
